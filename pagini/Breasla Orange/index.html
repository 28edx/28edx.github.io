Breasla Orange

Breasla Orange este cel mai actual grup (precedentele fiind: Dâmbovița și micro 2 târgovșite) cu membrii: 28edxwrd, edibos, David, Pol și MariusBDR.

Istoria Grupului

Grupul a fost co-fondat de către 28edxwrd și edibos în data de 08.12.2022 la ora 22:48 după ce s-a propus ideea creeării unui server nou de discord.


Primul mesaj de pe grup a fost trimis de către edibos, conținând un fișier numit „matrice.xlsx”.


Primul membru al grupului, care nu a fost unul dintre fondatori a fost David.


Breasla Orange avea în precedente diferite denumiri și nume. Din păcate acestea au fost pierdute cu timpul. Primul nume al serverului se presupune că a fost „cafelutsa bombardieră” și cu o poză cu David.


Serverul actual este unul din cel mai vechi server deținut de grup, cu un record în momentul scrierii de 1 an și 4 luni. Cel mai vechi este „micro 2 târgovșite” având 1 an și 9 luni iar „Dâmbovița” doar 8 luni.

Teoretic, prima instanță acestui grup datează din 11.11.2022 atunci când a fost creat grupul „cafelutsa bombardieră”.


Membrul Pol a intrat pe server după ce a fost contactat pe messenger de David și 28edxwrd.


Al 5-lea membru care a intrat a fost MariusBDR.


Pe lângă membrii menționați, a existat un al 6-lea membru Buba care nu se mai află pe server după cearta cu membrul Pol.
